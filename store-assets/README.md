# Chrome Web Store Assets

These files are prepared for the Chrome Web Store listing and are not bundled into the extension package.

Use:

- `public/icons/icon-128.png` as the store icon.
- `small-promo-440x280.png` as the required small promotional tile.
- `marquee-promo-1400x560.png` as the optional marquee promotional tile.
- `screenshot-feed-markers-1280x800.png`, `screenshot-settings-1280x800.png`, and `screenshot-popup-1280x800.png` as listing screenshots.

The screenshots use synthetic demo content. Do not replace them with images that expose real users, real feed content, API keys, or private account details.
